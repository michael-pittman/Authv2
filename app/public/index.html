Welcome to RDPDuck

A web app that brings Remote Desktop Connection/RDP to the browser

Duck, duck, RDP

RDP Duck brings RDP to the browser for a whole, new Remote Desktop Connection experience. Some features include:

  • Fast and easy way to create connections between your web broswer and your remote desktop
  • Instructions on how to turn any desktop into a remote desktop
  • Tools to grab needed info to setup your remote desktop
  • Chat portal
